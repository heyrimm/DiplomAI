**2026년 외교 공공데이터·AI 활용 경진대회 출품작 (제품·서비스 개발 부문)**

## 1. 제품·서비스의 목적 및 배경

### 1-1. 문제 정의 — 공공외교의 "설계 공백"

공공외교법(제2조)은 공공외교의 주체를 국가뿐 아니라 **지방자치단체 및 민간부문**으로 명시한다. 그러나 현장의 기획 도구는 이를 뒷받침하지 못하며, 이는 국가기관의 공식 진단으로 확인된다.

- **국회입법조사처**: 공공외교 추진체계의 문제로 ▲대상의 **지역별·국가별 통계 시스템 미비** ▲**평가체계 부재** ▲수행주체의 중앙부처 편중을 지적.
- **대한민국시도지사협의회**: 중앙·지자체·민간의 공공외교 활동이 상호 조율 없이 수행되어 **유사·중복·편중·누락**이 발생함을 보고.

실제로 본 서비스가 4개 기관 데이터를 교차 분석한 결과, **KOICA ODA는 활발히 진행되나 KF 공공외교 사업은 부재·중단된 국가가 23개국** 감지되었다. 국가기관이 지적한 '편중과 누락'이 데이터로 실증된다.

| 예시 국가 | KOICA ODA (2023, 라이브 검증) | 전년比 | KF 공공외교 사업 |
|---|---|---|---|
| 에티오피아 | **339.9억 원** | +30.2% | 부재·중단 (중단연도 [[검증 필요]]) |
| 우간다 | **252.6억 원** | +14.8% | 부재·중단 (중단연도 [[검증 필요]]) |
| 몽골 | **224.3억 원** | +27.4% | 부재·중단 (중단연도 [[검증 필요]]) |

*→ ODA는 매년 두 자릿수로 증가(활발)하는데 공공외교 채널은 비어 있는 전형적 '공백' 패턴. 개별 수치는 서비스 화면에서 라이브 데이터로 검증된 값만 표시한다.*

### 1-2. 대상 사용자 — 최우선 사용자를 한 명으로 특정

세 부류 모두 **소속 기관의 내부 시스템 접근권이 없어 MOFAI·KOICA·KF 내부 도구를 사용할 수 없다**는 공통 처지에 있으나, 본 서비스는 최우선 사용자를 **한 명으로 명확히 특정**하여 설계한다.

| 우선순위 | 사용자 | 처지 | 핵심 니즈 |
|---|---|---|---|
| **Primary (최우선)** | **지자체 국제교류 담당 공무원** | 순환보직·소수 인력, 예산 근거 요구가 가장 강함, 내부 데이터 도구 전무 | 어느 나라와·왜 교류할지 **근거 있는 대상국 선정 + 상신용 계획서** |
| Secondary | 대학 국제처 실무자 | 학술·한국학 교류 기획, 기관 포털 접근 제한 | 한국학 인프라 기반 협력 대상 발굴 |
| Secondary | 민간 재단·NGO 실무자 | 사업 제안·공모 대응, 데이터 취합 수작업 | 제안서용 데이터 근거·타당성 |

DiplomAI는 **Primary 사용자(지자체 국제교류 담당 공무원)의 '대상국 선정 → 상신 문서화' 동선**을 1차 기준으로 설계하고, 나머지 두 부류는 동일 엔진을 공유하는 보조 사용자로 수용한다.

### 1-3. 왜 지금까지 해결되지 못했나

1. **제도의 역사가 짧다** — 공공외교법 2016년 제정. 30년 이상 축적된 ODA 대비 통계·평가 체계화가 뒤처짐.
2. **데이터가 기관 사일로에 갇혀 있다** — 외교부·KOICA·KF·한아프리카재단·세종학당재단이 각기 다른 포털·인코딩(EUC-KR/UTF-8 혼재)으로 개방. 위 23개국 공백은 **어느 단일 기관 화면에서도 보이지 않았다.**
3. **기술적 한계** — 생성형 AI 이전에는 데이터 '조회'까지만 가능. 데이터에서 **사업계획서라는 실무 산출물**로 잇는 마지막 단계가 불가능했다.
4. **시장 실패 영역** — 수요자가 지자체·대학·비영리라 민간 개발 유인이 없고, 정부 AI 투자는 내부 효율화에 우선 배정됨.

### 1-4. 서비스의 목적

외교부는 2025년부터 '외교 속의 AI' 3개년 사업으로 생성형 AI(**MOFAI**)를 도입, 내부 직원의 보고서 작성 자동화를 추진한다. 그러나 이는 업무망 내 폐쇄형 플랫폼으로, 공공외교법상 주체의 나머지 절반인 지자체·민간에는 어떤 도구도 없다.

> **DiplomAI의 목적: 외교부가 MOFAI로 내부 직원에게 제공하기 시작한 'AI 기반 업무 혁신'을, 공공데이터만으로 공공외교 생태계 전체(지자체·대학·민간)에 개방하는 것.**

이는 원년을 맞은 **제3차 공공외교 기본계획(2026–2030)** 의 '통합적·체계적 공공외교 생태계 구축'을 데이터·AI로 뒷받침하는, 생태계의 첫 공용 인프라 제안이다.

## 2. 제품·서비스의 기능 및 특징

### 2-1. 서비스의 척추 — 감지 → 추천 → 계획서

DiplomAI는 대시보드가 아니라 **하나의 업무 동선을 관통하는 코파일럿**이다. 서비스의 등뼈는 세 개의 핵심 기능이며, 나머지는 모두 이 셋을 뒷받침하는 보조 기능이다.

> **① 공공외교 공백 감지 → ② 근거 인용 사업계획서 생성 → ③ 원클릭 파이프라인.**
> 첫 화면에서 공백 국가를 발견하고(감지), 클릭 몇 번으로 AI 추천을 받아(추천), 그중 하나를 고르면 KPI·리스크가 포함된 **사업계획서 초안이 생성·출력**된다(계획서). 대상국 조사에 수일이 걸리던 업무가 수 분으로 단축된다.

### 2-2. 핵심 3기능 (척추)

| # | 핵심 기능 | 내용 | 핵심 데이터 |
|---|---|---|---|
| **①** | **공공외교 공백 자동 감지** (첫 화면) | KOICA ODA 실적과 **KF 공공외교 사업 이력**을 전 국가 교차 스캔 → "ODA는 활발하나 공공외교가 부재·중단된 국가" **23개국을 자동 탐지**, 홈 최상단에 ODA 규모순 노출. 카드 클릭 시 국가 상세로 즉시 진입. | KOICA 지원실적 **× KF 사업정보** |
| **②** | **근거 인용 사업계획서 초안 생성** | 배경–목표–대상·수혜자–활동–예산–**KPI**–**리스크(여행경보 자동 반영)** 구조의 계획서 생성. 본문 인용 수치는 **하이라이트 + 출처 툴팁**([[수치\|출처]] 마커)으로 표시되어 문장 단위 검증 가능. Markdown·인쇄/PDF 출력. | 전체 융합 데이터 + 여행경보 API |
| **③** | **감지→추천→계획서 원클릭 파이프라인** | 추천 카드의 "이 사업으로 계획서 생성" 버튼으로 해당 추천이 계획서의 기반 사업으로 자동 연결 — 분석에서 실무 문서까지 **끊기지 않는 코파일럿 동선**. | UX 설계 |

※ ①과 ② 사이를 **AI 사업 추천**이 잇는다: 국가별 융합 데이터를 근거로 ODA 3건 + 공공외교 2건을 생성하며, 모든 추천에 인용 데이터 출처를 명시한다.

### 2-3. 보조 기능 (핵심 3기능을 뒷받침)

핵심 동선을 보강하는 진단·근거 도구로, **독립된 주기능이 아니다.**

| 보조 기능 | 역할 | 데이터 |
|---|---|---|
| **ODA 섹터 편차(사각지대) 진단** | *공공외교 공백(①, 국가 단위)과는 별개 개념.* 한 국가 **내부**에서 분야별 지원 '비율'을 동일 지역 평균과 실계산 비교, 지역 평균 대비 30% 이상 부족 분야를 표시하는 **진단 보조 지표**(하드코딩 없는 전량 실데이터 연산). | KOICA 지역별·분야별 CSV |
| **사업 타당성 진단** *(본 초안 이후 신규 추가)* | 사용자가 **자신의 사업 아이템·사업계획서(PDF 업로드 포함)** 를 넣으면, 공공데이터 신호(분야 적합도·개발니즈·ODA 채널·공공외교 연계·SDG)를 규칙 기반으로 합산한 **가능성 점수 + 강점·리스크·개선안**을 제시. 코파일럿을 '생성'에서 '**검증**'까지 확장. | 전체 융합 데이터 + Claude |
| 정책 시뮬레이션 | 분야별 예산 증감 시 HDI·수혜인구·SDG 기여 변화의 **추정치**(계수 근거 명시) + AI 정성 분석. | 협력국 통합 개발지표 |
| 국가 종합 대시보드 | ODA 시계열, 공공외교 채널 5종, 실시간 여행경보·안전공지, 아프리카 국가의 국내 지자체 교류 선례를 단일 화면 제공. | 전체 데이터 |

## 3. 외교부 및 산하기관 공공데이터 활용 방안

### 3-1. 활용 데이터 목록 — 주최·주관 4개 기관 전체 활용 (전량 data.go.kr 개방)

| 기관 | 데이터 | 형태 | 서비스 내 활용 |
|---|---|---|---|
| **외교부** | 여행경보 (Open API) | API | 홈 경보 현황 · 국가 대시보드 · **계획서 리스크 자동 반영** |
| **외교부** | 안전공지 (Open API) | API | 국가별 실시간 안전공지 |
| **외교부** | 재외동포현황 | CSV | 공공외교 채널 지표 · 추천 근거 |
| **외교부** | 재외공관 현황 | CSV/API | 공공외교 인프라 지표 |
| **KOICA** | 국가별 지원실적 (1991–2023 시계열) | CSV | ODA 추이·전년比 · **공백 감지** · 홈 Top10 |
| **KOICA** | 사업분야별·지역별 실적 | CSV | 분야 구성 · 사각지대 지역 평균 실계산 |
| **KOICA** | 협력국 통합 개발지표 | CSV | 분야별 비율 · 시뮬레이션 계수 |
| **KOICA** | SDG 성과지표 | CSV | 분야–SDG 태깅 |
| **KF** | 융합 KF-공공외교·KOICA-ODA 사업정보 **(11,782건)** | CSV | 국가별 KF 사업 이력 · **공백 감지 핵심 소스** |
| **KF** | 해외대학 한국학 과정 운영현황 **(1,421건)** | CSV | 한국학 인프라 지표 · 추천 근거 |
| **한·아프리카재단** | 지자체-아프리카 교류협력 사례 **(387건)** | CSV | 아프리카 국가 선택 시 국내 지자체 선례 제시 |
| 세종학당재단 (문체부 산하·유관) | 국가별 수강생 현황 | CSV | 한국어 학습 수요 지표 |

> **KF 데이터의 위상**: 본 서비스의 최대 차별점인 '공공외교 공백 감지'는 **KF '융합 공공외교·ODA 사업정보(11,782건)'를 핵심 소스**로, KOICA ODA 실적과 국가 단위 교차하여 산출된다. KF 데이터가 없으면 성립하지 않는 기능이며, 이는 KF의 데이터 개방이 창출하는 정책적 가치를 직접 실증한다.

### 3-2. 획득 방법

- 전량 **공공데이터포털(data.go.kr)** 오픈 API 인증키 발급 및 파일데이터 다운로드. 별도 협약·비공개 데이터 없음 → **누구나 재현 가능.**

### 3-3. 데이터 가공 (활용 범위)

- 기관별 상이한 인코딩(EUC-KR·CP949·UTF-8-SIG) **자동 판별 파이프라인**.
- 기관별 상이한 국가 표기(국문·영문·약칭) **매핑 테이블로 협력국 70개국 정규화.**
- 단순 재게시가 아닌 **파생 지표 생산**: ▲기관 교차 공백 신호(23개국) ▲지역 평균 분야 비율(섹터 편차) ▲전년比 증감률 ▲공공외교 채널 지수(산식·가중치 문서화).
- AI 생성물의 모든 인용 수치가 화면에서 원천 데이터 출처와 연결되는 **추적 가능성(traceability)** 구현.

### 3-4. 확보의 지속성

- 여행경보·안전공지는 **실시간 API**로 상시 최신성 유지.
- CSV는 각 기관 연 단위 정기 갱신 대상으로, **파일 교체만으로 갱신**되는 구조(파일명 패턴 매칭 로더).
- 향후 KOICA 오픈데이터포털 API·외교부 LOD 연계로 자동 갱신 확대 예정.

## 4. 기존 제품(서비스)와의 차별성 및 독창성

### 4-1. 첫 번째 독창 요소 — 기관 교차 공백 감지 (리드)

DiplomAI의 근본 독창성은 **서로 다른 기관(KOICA × KF) 데이터를 국가 단위로 교차하여, 단일 기관 포털 화면에는 절대 보이지 않는 정책 공백 23개국을 자동 탐지**한다는 점이다. 이는 데이터 '조회'가 아니라 데이터 '융합'으로만 가능한 인사이트이며, 첫 화면 최상단에 노출되어 사용자의 의사결정을 즉시 이끈다.

### 4-2. 본 대회 기존 수상작과의 차별성

'ODA 데이터 기반 사업 발굴'은 **2023년(수원국 개발수요 파악)·2024년(CSO 사업역량 지원) 최우수상**이 이미 다룬 영역이다. DiplomAI는 세 지점에서 이를 넘어선다.

| 구분 | 기존 수상작(ODA 사업 발굴) | **DiplomAI** |
|---|---|---|
| **(a) 도메인** | ODA(개발협력) | **공공외교 설계** — 본 대회 수상 이력이 없고, 국가기관이 도구 부재를 공식 지적한 영역 |
| **(b) 산출물** | 분석·발굴 결과 제시 | **KPI·리스크 포함 실무 사업계획서** — '아이디어'가 아닌 '업무 문서' |
| **(c) 완성도** | 기획·분석 | **배포되어 작동하는 제품 + 전량 공개 데이터 오픈소스** — 누구나 재현 가능 |

### 4-3. 기존 서비스와의 비교

| 기존 | 한계 | DiplomAI |
|---|---|---|
| KOICA 오픈데이터포털 | ODA 데이터 조회에서 종료 | 조회 → 진단 → 추천 → **계획서 생성**까지 전 과정 |
| KF 통계센터 | 단일 기관 데이터 조회 | 4개 기관 **교차 융합** — 단일 화면에 없는 공백 23개국 감지 |
| 외교부 MOFAI (내부 시범) | 내부 직원 전용·폐쇄망 | 공공데이터 기반 **공개 서비스** — 지자체·민간 등 법정 주체 커버 |
| 반크 AI 상황실 (89개국 인식 모니터링) | 외부 '인식' 데이터 중심 | 우리의 '사업 자산' 데이터 기반 **설계 도구**(상호보완) |

### 4-4. 독창적 기술 요소

- **기관 교차 공백 감지 알고리즘** — KOICA×KF 국가 단위 교차로 공백 23개국 자동 탐지·홈 노출. 융합으로만 가능한 인사이트.
- **근거 강제·근거 가시화 생성 설계** — AI가 제공된 공공데이터 수치를 2개 이상 직접 인용해야만 계획서를 생성하도록 프롬프트를 구조화(환각 억제)하고, 생성문의 모든 인용 수치를 **하이라이트+출처 툴팁**으로 시각화. "AI의 주장"이 아닌 "공공데이터의 근거"임을 문장 단위로 검증 가능.
- **감지→추천→계획서 원클릭 파이프라인** — 공백 발견에서 실무 문서 출력까지 단절 없는 동선. 대시보드가 아닌 **코파일럿**.
- **비율 기반 섹터 편차 방법론** — 총액 규모차에 왜곡되지 않는 지역 평균 '비율' 비교로 방법론적 타당성 확보.

## 5. 기대효과

### 5-1. 사회적 가치 (상생협력·공공성)

- **지역균형 (S)** — 수도권 대비 열악한 지자체 국제교류 조직의 공공외교 기획 역량 격차를 도구로 해소. 시도지사협의회가 지적한 지자체 지원 공백에 대한 실질적 응답.
- **행정 효율·투명성 (G)** — 국회입법조사처가 지적한 유사·중복·편중을 데이터 교차 감지로 예방 → 예산 중복 집행 방지. 모든 AI 산출물이 출처와 함께 표시되는 **근거 기반(evidence-based) 행정** 지원.
- **SDG 연계 (E·S)** — 추천·계획서에 SDG 자동 태깅, SDG 17(글로벌 파트너십) 이행을 데이터로 지원.

### 5-2. 정책 기여

- **제3차 공공외교 기본계획(2026–2030)** '통합·체계적 생태계 구축'의 이행 도구 — 원년에 제안하는 생태계 공용 인프라.
- 외교부 AI 3개년 사업의 **대국민 맞춤형 AI 단계에 연계 가능한 선행 프로토타입** — 공공데이터만으로 실현 가능성 검증.
- **정책 공백의 실증 발견** — '공백 23개국' 목록 자체가 공공외교 자원 배분 재검토의 기초 자료.
- **데이터 개방 선순환** — 개발 과정에서 확인한 개방 확대 필요 데이터(KF 사업별 예산·성과의 기계가독형 개방, 지자체 국제교류 표준 데이터셋)를 역제안.

### 5-3. 경제적 효과·일자리

- 대상국 조사·기획 초안 작성에 소요되던 **수일의 업무를 수 분으로 단축** → 소수 인력 조직의 실질 기획 역량 증대.
- 공공외교·국제개발 분야의 **데이터 기반 사업기획 직무** 창출 및 관련 창업 생태계 자극.

## 6. 제품·서비스의 사업(창업) 계획 — 예비창업자 관점

### 6-1. 단계별 로드맵

**1단계 — 공개·검증 ('26 하반기)**
- 웹서비스 무료 공개·오픈소스 유지로 레퍼런스 확보.
- **Primary 사용자(지자체 국제교류 부서)** 중심 파일럿 10곳 확보, 사용성 피드백 기반 고도화(대학 국제처·재단은 보조 검증).
- 외교부·KF 데이터 개방 확대 제안 등 생태계 활동 병행.

**2단계 — 수익화 ('27)**
- **B2G/B2I 구독**: 지자체·대학·재단 기관용 플랜(국가 심층 리포트 자동 생성, 담당 교류국의 여행경보 변경·KF 사업 신규 등록·공백 신호 발생 시 맞춤 알림).
- **컨설팅 리포트**: 신규 교류국 발굴·자매결연 후보 분석 등 건별 데이터 리포트.
- 조달청 혁신제품·벤처나라 등록으로 공공조달 판로 확보.

**3단계 — 확장 ('28~)**
- 외교부 AI 플랫폼 대국민 단계와의 연계 협력 제안(공공데이터 기반 민간 프론트엔드 모델).
- 데이터 확장: 국가별 한국 인식 조사·문화행사·한류 데이터 결합으로 '**설계(DiplomAI) + 인식 모니터링**' 통합 플랫폼화.
- 해외 공여기관 대상 현지화 수출 모델 검토.
